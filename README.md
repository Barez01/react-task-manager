# Blue Planet

A High level React application built for managing daily tasks. Written in typescript and has full clean code and architecute.

## Features
- Built with React 
- Styled with CSS Modules & Tailwind
- Modular component structure
- Basic CRUD operations
- Auhtorization with username and password

## Installation

1. **Clone the backend**  
   ```sh
   get access through: https://github.com/Barez01/node_tasks.git
2. **Clone the repository**  
   ```sh
   git clone https://github.com/Barez01/react-task-manager.git
3. **Navigate to the project directory**  
   ```sh
   cd react-task-manager
4. **Install dependencies**  
   ```sh
   npm install
5. **Start the development server**  
   ```sh
   npm start
